Simple Music Widget creates a widget that can be used to display a music player with details.

== Description ==
Simple Music Widget is a plugin which creates a widget that can be used to display a music player with artist's name, song, and a cover image. The user is also able to add a short description about the song.

= Features =
* Easy to use audio players
* Supports HTML5 audio player
* Supports WordPress default audio embed
* Fully responsive (your Theme must be responsive for this to work)
* Customize the player via custom css
* Upload cover image and audio files
* Use external cover image url and audio file url
* Compatible with Page Builder by SiteOrigin

= Requirements =
Your Theme must support widgets in order to use this plugin. Visit [WordPress Widgets](http://codex.wordpress.org/WordPress_Widgets) for more details. Also, it only works with browsers which support HTML5 audio element.

= Translations =
To translate the plugin to a particular language, use the file provided in /languages/ folder.

== Installation ==

= From your WordPress dashboard =
1. Visit 'Plugins > Add New'
2. Search for 'simple music widget'
3. Activate Simple music widget from your Plugins page.
4. Go to Appearance > Widgets
5. Drag and drop 'simple music widget' in the widget area
6. Configure the widget

= From WordPress.org =
1. Download and upload the 'simple-music-widget' directory to /wp-content/plugins/ directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Appearance > Widgets
4. Drag and drop 'simple music widget' in the widget area
5. Configure the widget

== Frequently Asked Questions ==

= Can I upload music/image files directly from my website? =
Yes, you can upload music files directly from the widget, just make sure the image you select is around 100x100 pixels and your audio file is mp3.

= Can I use external audio file and photo? =
Yes, just copy and paste the direct url to the desired field.

= Can I have multiple widgets? =
Yes, you can have multiple widgets with different songs.

= What is the "Use WordPress Audio Shortcode" option? =
Since version 1.2, users are able to select their desire method for playing audio files. You can use the default HTML5 player or choose to use the WordPress Audio Shortcode to play the audio.
