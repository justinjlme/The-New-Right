Best tool for importing, merging and displaying Curated RSS and Atom feeds on your WordPress site.


== Description ==

Using the CM Curated RSS Aggregator you can create unlimited amount of curated lists of links grouped by categories and tags.
Each link has its own description and favicon. Lists have easy search and filtering utilities.


== Installation ==

1. Upload the plugin folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure plugin.
4. Add lists and categories.
5. Use shortcode to display list.
